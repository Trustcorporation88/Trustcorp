# Trust Corporation Hub

Site holding da TrustCorp (`www.trustcorp.com.br`).

## Deploy no Railway

1. Conecte este repositório no Railway
2. Branch: **`main`**
3. Root Directory: deixe vazio (raiz do repo)
4. Build: automático (Dockerfile ou Nixpacks)
5. Generate Domain

### Start command
```bash
npm start
```

### Domínio
- Custom domain: `www.trustcorp.com.br`
- DNS: CNAME `www` → domínio gerado pelo Railway
